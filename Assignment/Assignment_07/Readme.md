# Assignment 7

This folder contains the following warm-up programs written in C.
All programs run based on user input and display specific numerical patterns.

Program List (In Sequence)

program7_1.c – Accepts a number and prints $ and * symbols that many times

How to Execute the Programs

program7_2.c – Accepts a number from the user and prints all numbers from 1 to that number

program7_3.c – Accepts a number and prints a number line from –N to +N

program7_4.c – Accepts a number and prints all odd numbers up to that number

program7_5.c – Accepts a number and prints the first 5 multiples of that number



Use the following commands to compile and run the C programs:

Compile
gcc filename.c -o myexe

Run
myexe
