# Assignment 68 : Recursive Pattern Programs

## Aim 
 
To study and implement **recursive functions in C** for printing different patterns without using any looping constructs.

---

## File Structure

```
Assignment_68/
│
├── program68_1.c   // Recursive program to display '*'
├── program68_2.c   // Recursive program to display numeric pattern (1, 2 3)
├── program68_3.c   // Recursive program to display reverse numeric pattern
├── program68_4.c   // Recursive program to display character pattern (F to A)
├── program68_5.c   // Recursive program to display alphabet pattern (a to f)
└── README.md
```

---

## Problem Statements

### Program 68_1

Write a recursive program which displays the following pattern.

**Output:**

```
*  *  *  *  *
```

**Prototype:**

```c
void Display();
```
